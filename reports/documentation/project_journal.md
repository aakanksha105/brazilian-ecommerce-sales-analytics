# Project Journal

## Why this project?

I wanted to build an end-to-end analytics project that demonstrates SQL, Python, PostgreSQL, Tableau, and business analytics skills relevant to Data Analyst and BI Analyst roles.

## Dataset Selection

I selected the Olist Brazilian E-commerce Public Dataset because it provides real transactional data across customers, orders, products, payments, reviews, and sellers. Marketing, session, and A/B testing data will be generated later to simulate real business analytics scenarios.

## Project Goals

- Build a relational PostgreSQL database
- Perform customer analytics
- Analyze marketing performance
- Measure retention
- Build executive dashboards
- Practice statistical testing
